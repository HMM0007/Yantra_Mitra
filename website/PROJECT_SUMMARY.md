## 🎛️ Customization Guide

### Change Logo
Replace `public/logo.svg` with your logo, or update the logo component in navbar:
```jsx
<img src="/your-logo.png" alt="YantraMitra" className="w-10 h-10" />
```

### Update Team Members
Edit team array in `TeamSection()` in App.jsx:
```javascript
const team = [
  { name: 'Your Name', role: 'Your Role', github: '#', linkedin: '#' },
  // Add more members
];
```

### Modify Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  sage: { /* custom sage shades */ },
  forest: { /* custom forest shades */ }
}
```

### Change Fonts
Update in `tailwind.config.js`:
```javascript
fontFamily: {
  sans: ['YourFont', 'system-ui', 'sans-serif'],
}
```
